# Stelat - Attendance Tracker

A professional, high-fidelity in-plant attendance and overtime tracking system developed for Stellantis.

## Features

- **Real-time Attendance Tracking**: Capture and monitor worker attendance with precision.
- **Overtime Management**: Track stay-back hours and calculate "OT Days" credits.
- **Briefing Analytics**: Dashboard and reporting for daily briefings.
- **Worker Management**: Detailed profiles and department-based organization.
- **Premium UI/UX**: Built with React, Tailwind CSS, and shadcn/ui for a professional, glassmorphic corporate aesthetic.
- **Secure Authentication**: Integrated with Supabase for robust identity management.

## Tech Stack

- **Frontend**: React 18 with Vite
- **Styling**: Tailwind CSS & Vanilla CSS
- **UI Components**: shadcn/ui & Lucide React
- **Backend**: Supabase (Auth & Database)
- **State Management**: TanStack Query (React Query)
- **Charts**: Recharts

## Getting Started

1. Clone the repository.
2. Install dependencies: `npm install`
3. Set up environment variables in `.env` (Supabase URL and Anon Key).
4. Run development server: `npm run dev`
5. Build for production: `npm run build`
